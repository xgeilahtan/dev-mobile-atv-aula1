### 📦 Instalação
Antes de executar, certifique-se de instalar as dependências em cada pasta (caso ainda não tenha feito):

```bash
# Instalar dependências do Ex02 e Ex03
cd ex02 && npm install && cd ../ex03 && npm install && cd ..
```

### 🚀 Como executar
#### Exercício 1: Funções em JavaScript
Execute a partir da raiz do projeto:

```bash
node ex01/index.js
``

#### Exercício 2: ArrayUtils com TypeScript
Este exercício utiliza módulos ES (ts-node-esm).

```bash
cd ex02
npx ts-node-esm arrayUtils.ts
(Para voltar à raiz após testar: cd ..)
```
#### Exercício 3: Pokedex CLI
Este exercício aceita um argumento (nome ou ID do Pokémon).

```bash
cd ex03
npx ts-node pokedex.ts pikachu
(Você pode substituir "pikachu" por qualquer outro Pokémon ou ID, ex: npx ts-node pokedex.ts 6)
```
